# 3D Math for Games and Visualization

A comprehensive, beginner-friendly guide to the mathematics behind 3D games, simulations, and visual applications.

Each chapter builds on the previous one, starting from the basics and working up to advanced topics. Every concept includes plain-English explanations, visual diagrams, and practical code-style examples so you can see exactly how the math applies to real game scenarios.

## How to Read

- **Start from Chapter 01** if you are new to 3D math.
- **Jump to any chapter** if you already know the basics and want a specific topic.
- Each chapter is self-contained with cross-references where needed.

## Conventions

- Vectors are written in **bold**: **v** = (x, y, z)
- Matrices are written in **uppercase bold**: **M**
- Scalars are written in *italics*: *t*, *d*, *s*
- Code examples use pseudocode that maps closely to C++, Python, or GLSL.

## Prerequisites

- Basic algebra (variables, equations, order of operations)
- A curiosity about how games work under the hood
